## 🚀 Tecnologias e práticas utilizadas:
<div align="center">

✔️ ASP.NET Core com .NET 6

✔️ Entity Framework Core

✔️ SQL Server

✔️ Swagger

✔️ Injeção de Dependência

✔️ Programação Orientada a Objetos
